# 员工所属民族分析面板 - UI 知识文档

## 1. 项目概览

员工民族分析面板（Ethnicity Analysis Dashboard），用于展示企业员工的民族构成分布，帮助 HR 了解组织多元化和包容性水平。

## 2. 布局与排版

### 2.1 整体页面结构
```
┌────────────── 顶栏 (Header) ──────────────┐
├───────────── 面包屑导航 ──────────────────┤
├───────────── 页面标题区 ──────────────────┤
├───────────── 筛选栏 (Filter Bar) ─────────┤
├───── 统计卡片 (Stat Cards, 4列) ─────────┤
├─── 饼图 ────────┼─────── 柱状图 ─────────┤
├──────── 热力图 (全宽, 部门×民族) ────────┤
├───────────── 数据明细表格 ───────────────┤
├───────────── 附注说明 ───────────────────┤
└────────────── 页脚 (Footer) ──────────────┘
```

### 2.2 断点与响应
- 桌面（≥1200px）：4 列统计卡片，2 列图表并排
- 平板（768-1199px）：2 列统计卡片，图表堆叠
- 手机（<768px）：1 列，侧边栏隐藏

### 2.3 间距规范
- 页面左右边距：32px（桌面），16px（移动端）
- 卡片间距：16px
- 卡片内边距：20px
- 标题与内容间距：16px

## 3. 色彩体系

### 3.1 主色调
| 用途 | 色值 | CSS 变量 / 备注 |
|------|------|----------------|
| 顶栏背景 | `#1a3a5c` → `#2d5f8a` | 渐变 |
| 强调色 | `#2d5f8a` | 按钮、图标、链接 |
| 深色文字 | `#1a3a5c` | 主标题、数值 |
| 常规文字 | `#333` | 正文 |
| 辅助文字 | `#888` / `#999` | 标签、副信息 |

### 3.2 图表颜色（民族区分）
- 汉族：`#1890ff`（蓝色）
- 回族：`#fa8c16`（橙色）
- 满族：`#722ed1`（紫色）
- 蒙古族：`#13c2c2`（青色）
- 壮族：`#52c41a`（绿色）
- 维吾尔族：`#eb2f96`（玫红）
- 藏族：`#faad14`（金黄）
- 其他使用 Ant Design 色板

### 3.3 民族徽章
- 汉族：`<span class="badge-han">` → 浅蓝底蓝字
- 少数民族：`<span class="badge-minority">` → 浅橙底橙字
- 其他：`<span class="badge-other">` → 灰底灰字

## 4. 组件规范

### 4.1 统计卡片 (StatCard)
- 白色背景，圆角 10px，轻微阴影
- 悬停时上移 2px，阴影加深
- 内容：图标 + 标签 + 大数字 + 趋势副文本
- 常用在页面顶部，4 列网格

### 4.2 图表卡片 (ChartCard)
- 与统计卡片同风格
- 标题栏含图标 + 标题 + 额外信息
- 图表区域固定高度 360px（全宽 460px）
- 使用 ECharts 渲染

### 4.3 筛选栏 (FilterBar)
- 白色背景，圆角 10px
- 水平排列标签 + 下拉选择器
- 部门、地区、统计周期三项筛选

### 4.4 数据表格
- Ant Design 风格：浅灰表头、斑马条纹悬停
- 行高 42px（含 padding）
- 列包含：序号、民族标签、人数（粗体）、占比（带进度条）、部门数、平均司龄、性别占比、趋势图标
- 顶部可搜索

### 4.5 进度条 (ProgressBar)
- 固定宽度 100px，高度 8px
- 圆角 4px
- 填充色渐变 `#2d5f8a → #5b8cb8`
- 宽度按占比百分比动态设置

## 5. 交互行为

### 5.1 图表交互
- 饼图：悬停显示 `民族名: 人数 (占比%)`，可点击扇区高亮
- 柱状图：悬停显示详细 tooltip
- 热力图：悬停显示 `部门 · 民族` + 人数，色块上有标签数字

### 5.2 筛选联动
- 切换部门/地区/周期 → 全局数据重新渲染（当前为模拟，后续对接 API）

### 5.3 导出刷新
- 「导出报告」按钮：可导出为 PDF/Excel（待实现）
- 「刷新数据」按钮：重新请求最新数据（待实现）

## 6. 数据字段约定（数据库对接）

### 6.1 核心指标
| 指标 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 员工总数 | total_employees | int | 当前在职 |
| 民族数 | ethnic_group_count | int | 涵盖不同民族数量 |
| 少数民族人数 | minority_count | int | 非汉族员工人数 |
| 少数民族占比 | minority_ratio | float | minority_count / total |
| 多样性指数 | diversity_index | float | Shannon 指数 |

### 6.2 民族分布数据
| 字段 | 类型 | 说明 |
|------|------|------|
| ethnic_name | string | 民族名称 |
| employee_count | int | 该民族人数 |
| ratio | float | 占比 |
| dept_count | int | 分布在多少个部门 |
| avg_tenure | float | 平均司龄（年） |
| male_ratio | float | 男性占比（0-1） |
| female_ratio | float | 女性占比（0-1） |
| trend_direction | string | up/down/flat |
| trend_value | float | 变化百分比 |

### 6.3 部门×民族交叉表
- 行：部门名称
- 列：民族名称
- 值：人数（int）

## 7. 技术栈

- **框架**：React + TypeScript
- **UI 组件库**：Ant Design 5.x
- **图表库**：ECharts 5.x
- **构建工具**：Vite 5.x
- **状态管理**：React hooks（useState + useEffect）
- **HTTP 请求**：axios 或 fetch
